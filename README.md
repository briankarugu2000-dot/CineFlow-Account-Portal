# CineFlow Account Portal

## Project Brief
CineFlow is a fictional video streaming platform. This project is a "New Content Creator" 
registration page that collects user data, validates it before submission, 
and features a promotional video and audio instructions.

## Technologies Used
- HTML5

## Features
- Registration form with multiple input types
- HTML5 form validation (required, pattern, minlength, accept)
- Embedded promotional video (MP4 and WebM formats)
- Audio tips section
- Grouped form fields using fieldset and legend
- Fully labelled inputs for accessibility

## Form Fields
- Full Name (text)
- Email Address (email)
- Password (password with minlength of 8)
- Date of Birth 
- Profile Picture upload (jpg and png only)
- Username (alphanumeric only using pattern validation)
- Content Genres (checkboxes)

## Setup Instructions
1. Clone the repository
2. Open index.html in your browser
3. No installation needed
